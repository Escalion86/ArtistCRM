# AGENTS.md — руководство для Codex

## Описание проекта

Это CRM, в которой также создается и приложение для Артистов.

## Язык

Используй в ответах Русский язык

## Структура

- /app: Исходные страницы
- /app/api: API
- /components: Компоненты
- /schemas: БД
- /models: модели БД (на текущий момент не используются)
- /layouts: крупные компоненты, модальные окна, карточки, листы, контент
- /svg: svg файлы
- /state: атомы и селекторы jotai, а также хелперы загрузок
- /server: сурверные компоненты
- /helpers: вспомогательные функции

## Стиль кода

- Следуйте принятому стилю и правилам именования

## Работа с компонентами

- Используйте React-функциональные компоненты и Next.js
- Применяйте проп-тайпинг

## Запреты для ИИ

- Запрещено создавать бинарные файлы, например картинки или аудио

## Комментрарий

- Если делаешь кнопки, то курсор при наведении должен быть pointer
