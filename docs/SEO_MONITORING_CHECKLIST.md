# SEO monitoring checklist

Дата старта: 2026-04-29

## Еженедельно

- Google Search Console:
  - проверить `Indexing -> Pages`: нет ли новых ошибок индексации;
  - проверить `Sitemaps`: `https://artistcrm.ru/sitemap.xml` успешно прочитан;
  - проверить `Performance -> Search results`: показы, клики, CTR, средняя позиция;
  - отдельно смотреть запросы: `crm для артистов`, `crm для ведущих`, `crm для музыкантов`, `заявки tilda crm`.
- Яндекс Вебмастер:
  - проверить `Индексирование -> Страницы в поиске`;
  - проверить `Диагностика -> Диагностика сайта`;
  - проверить, что в индексе нет `/login`, `/cabinet`, `/event/*`;
  - проверить статус sitemap и дату последней загрузки.
- Яндекс Метрика:
  - отчет `Источники -> Поисковые системы`;
  - цели регистрации/входа/перехода на тарифы/входящей заявки;
  - страницы входа из поиска и показатель отказов.

## Ежемесячно

- Core Web Vitals:
  - LCP на мобильных;
  - CLS;
  - INP;
  - страницы с худшими показателями в Google Search Console.
- Контент:
  - проверить, какие посадочные дают показы, но имеют низкий CTR;
  - обновить title/description для страниц с CTR ниже 2%;
  - добавить 1-2 новых посадочных страницы только под реальные запросы.
- Индексация:
  - сверить sitemap со списком публичных страниц;
  - убедиться, что canonical всех публичных страниц ведет на `https://artistcrm.ru`;
  - проверить, что `www.artistcrm.ru` и `http://artistcrm.ru` редиректят на `https://artistcrm.ru`.

## KPI

- В индексе Google и Яндекса есть главная и все SEO-посадочные.
- В индексе нет приватных страниц кабинета.
- Растут показы по небрендовым запросам.
- CTR по основным посадочным не ниже 2-3%.
- Нет красных Core Web Vitals на мобильных.
- В Метрике видны заявки/регистрации из органического поиска.

## После каждого SEO-релиза

1. Открыть `https://artistcrm.ru/robots.txt` и проверить sitemap/host.
2. Открыть `https://artistcrm.ru/sitemap.xml` и проверить список URL.
3. Проверить HTML главной: canonical, description, `og:image`.
4. Отправить sitemap на переобход в Яндекс Вебмастере.
5. В Google Search Console запросить проверку главной и новой посадочной страницы.
