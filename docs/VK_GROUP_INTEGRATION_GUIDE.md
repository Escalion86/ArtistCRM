# VK Group Integration Guide

Интеграция VK принимает входящие сообщения из группы VK, создает заявки в
ArtistCRM и сохраняет переписку по каждому диалогу.

Настройка выполняется индивидуально для каждого пользователя ArtistCRM в
`Настройки -> Интеграции -> VK`.

## Что нужно со стороны VK

1. Группа VK, где включены сообщения сообщества.
2. Токен сообщества с правами на сообщения.
3. Callback API для группы.
4. Строка подтверждения сервера Callback API.
5. Secret key Callback API.

## Подключение

1. В ArtistCRM откройте `Настройки -> Интеграции -> VK`.
2. Укажите `VK Group ID`.
3. Укажите токен сообщества VK.
4. Укажите строку подтверждения Callback API.
5. Нажмите `Подключить` в ArtistCRM.
6. Скопируйте `Адрес webhook VK`.
7. Скопируйте `Secret key Callback API`.
8. В управлении группой VK откройте `Работа с API -> Callback API`.
9. Добавьте сервер, вставьте webhook, secret key и подтвердите сервер.
10. В типах событий включите входящие сообщения.

## Что происходит при новом сообщении

1. VK отправляет событие `message_new` в ArtistCRM.
2. ArtistCRM находит пользователя по персональному webhook token.
3. ArtistCRM проверяет `secret` и `group_id`.
4. Первое сообщение из нового `peer_id` создает заявку со статусом `draft`.
5. Источник заявки: `VK`.
6. Переписка сохраняется в `VkConversations`.
7. Все сообщения сохраняются в `VkMessages`.
8. Повторные сообщения из того же `peer_id` не создают дубликаты заявок.

## Переписка

В карточке VK-заявки и карточке клиента доступен блок `Переписка VK`.

Из CRM можно:

- видеть историю входящих сообщений;
- отправлять ответ в VK;
- сохранять статус отправки сообщения;
- вручную привязывать conversation к клиенту или заявке через защищенный API.

## Безопасность

- Токен сообщества хранится в персональных настройках пользователя.
- Webhook URL содержит персональный token ArtistCRM.
- Callback API `secret` проверяется на каждом входящем событии.
- Все выборки и записи ограничены `tenantId`.
- В логах не должны появляться токены VK и полные payload с персональными
  данными без необходимости.
