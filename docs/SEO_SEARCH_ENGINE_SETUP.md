# SEO setup for Yandex and Google

Дата: 2026-04-27

## Что сделать после деплоя

1. Проверить production-домен:
   - основной домен открывается по HTTPS;
   - `http -> https` редиректит;
   - `www` и non-`www` приведены к одной версии;
   - canonical на главной совпадает с основным доменом;
   - `/sitemap.xml` открывается и не содержит `/login`, `/cabinet`, `/event/*`.

2. Подключить Яндекс Вебмастер:
   - добавить основной домен;
   - подтвердить права через DNS TXT или HTML-файл;
   - отправить `https://artistcrm.ru/sitemap.xml`;
   - проверить разделы "Индексирование", "Диагностика", "Страницы в поиске";
   - настроить региональность: Россия, если продукт продвигается в РФ.

3. Подключить Google Search Console:
   - добавить ресурс типа Domain;
   - подтвердить через DNS TXT;
   - отправить sitemap;
   - проверить "Pages", "Sitemaps", "Core Web Vitals", "Enhancements".

4. Подключить аналитику:
   - Яндекс Метрика с целями: регистрация, вход, клик по тарифам, отправка заявки/лида;
   - Google Analytics 4 или другой источник, если нужен Google organic tracking;
   - UTM-разметка для внешних размещений.

5. Собрать семантику:
   - `crm для артистов`;
   - `crm для ведущих`;
   - `crm для музыкантов`;
   - `учет заявок артист`;
   - `учет оплат мероприятий`;
   - `crm с google календарем`;
   - `заявки с tilda в crm`;
   - `договор для артиста crm`.

6. Сделать первые посадочные страницы:
   - `/crm-dlya-artistov`;
   - `/crm-dlya-vedushchih`;
   - `/crm-dlya-muzykantov`;
   - `/crm-dlya-tilda-zayavok`;
   - `/crm-s-google-calendar`.

## Как понять, что SEO работает

- Sitemap принят без ошибок.
- В индексе есть главная, политика, соглашение и будущие посадочные страницы.
- В индексе нет `/login`, `/cabinet`, `/event/*`.
- В Search Console появляются показы по брендовым и продуктовым запросам.
- В Яндекс Вебмастере нет критичных ошибок обхода.
- Core Web Vitals на мобильных в зеленой или желтой зоне, без красных LCP/CLS.

## Контентные правила для посадочных

- Один основной запрос на одну страницу.
- Уникальный `title`, `description`, `h1`.
- Первый экран сразу отвечает на запрос, без общих фраз.
- На странице должны быть: сценарии использования, список функций, FAQ, тарифный CTA, внутренняя ссылка на главную.
- Не копировать один и тот же текст между страницами, иначе Яндекс и Google будут считать страницы дублями.
