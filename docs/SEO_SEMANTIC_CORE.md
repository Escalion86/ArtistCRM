## Правила расширения ядра

- Не смешивать несколько разных поисковых намерений на одной странице.
- Для каждого кластера делать уникальные `title`, `description`, `h1`, FAQ и примеры сценариев.
- Сначала продвигать низко- и среднечастотные запросы с явной коммерческой потребностью.
- После подключения Яндекс Вебмастера и Google Search Console сверить фактические показы с этим ядром и расширить список.
